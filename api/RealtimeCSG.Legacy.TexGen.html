    Declaration
    public uint SmoothingGroup
    Field Value
    UInt32

    SmoothingGroup used to smooth normals together
    Remarks

    All normals around a vertex are averaged for all surfaces that have the same SmoothingGroup. When SmoothingGroup is 0 no smoothing is applied.

    Struct BrushMeshInstance

    Represents the instance of a BrushMesh. This can be used to assign a BrushMesh to one or multiple CSGTreeBrushes.

    Syntax
    public struct BrushMeshInstance
    Remarks

    See the Brush Meshes article for more information.

    note

    Be careful when keeping track of BrushMeshInstances because BrushMeshIDs can be recycled after being Destroyed.

    Struct CSGPlane

    A geometric plane

    Syntax
    [Serializable]
public struct CSGPlane
    Remarks
    note

    This code will be replaced by UnityEngine.Plane eventually. In older versions of Realtime-CSG UnityEngine.Plane, Unity's own Plane, was not serializable and wasn't an option.

    note

    This class can be safely serialized.
